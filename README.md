# Library Management System
The Library Management System is a console-based application developed in C++ using Object-Oriented Programming (OOP) concepts. It is designed to simplify the management of library operations by maintaining book records and handling borrowing activities efficiently.
## Features
- Add Book
- Display All Books
- Search Book by Title or Author
- Issue Book
- Return Book

## Language
C++

## Concepts Used
- Classes
- Objects
- Arrays
- Loops
- Conditional Statements
- Functions
